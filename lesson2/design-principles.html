PARC: Repetition

Nestle

https://www.nestle.com/

On this home page, the name of the company is repeated in the logo and the picture with the first name in the cups. It is a proper repetition of elements that provides a sense of a cohesive relationship between those elements. A design that repeats the same navigation structure, layout, schemes, colors, styles, and fonts throughout a site provides a sense of belonging and enhances the user experience.

PARC: Contrast

Siemens

https://new.siemens.com/ar/es.html

On this home page, there is a beautiful contrast between colors and fonts. It has distinct differences between foreground and background colors in order to enhance readability. Contrast does not only deal with colors but also blends into a visual hierarchy of font sizes, image placement, and content. Through the contrast, a focal point and a visual hierarchy are created at the main bottom to know more about the company.

Visual Hierarchy

Unilever

https://www.unilever.com/

On this page, the visual hierarchy is obtained through the button center, picture, and the different colors around this. All these elements point to this button and invite you to know more about the result. Visual hierarchy is accomplished using proportion, color, images, contrast, etc. The purpose of visual hierarchy is to provide a focal point; this indicates to visitors where the most critical information is located.
